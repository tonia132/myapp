# 🧪 MyApp 前端系統 (Vue 3 + Vite + Element Plus)

這是測試管理系統的前端專案，採用 **Vue 3 + Element Plus + Vite** 建構，  
搭配後端 Express + Sequelize API 使用。

---

## 🚀 開發環境需求

- Node.js 20+
- npm 10+
- 已安裝 MySQL / 後端伺服器運行中

---

## 📦 安裝與啟動

```bash
# 安裝相依套件
npm install

# 啟動開發伺服器
npm run dev

# 打包正式版
npm run build

# 檢視正式版 (本地伺服器)
npm run preview
